# 🏛️ O'zbekiston Milliy Universiteti Jizzax Filiali

![O'zMU Jizzax filiali](public/images/hero-bg.png)

## 🌐 Loyiha haqida
Ushbu loyiha **Mirzo Ulug'bek nomidagi O'zbekiston Milliy universiteti Jizzax filiali**ning rasmiy frontend platformasi hisoblanadi. Loyiha zamonaviy ta'lim muassasasi uchun zarur bo'lgan barcha raqamli imkoniyatlarni o'z ichiga oladi va foydalanuvchilarga yuqori sifatli (premium) dizayn hamda tezkorlikni kafolatlaydi.

---

## ✨ Asosiy Imkoniyatlar

### 🎨 Premium Dizayn va UX
*   **Modern Aesthetics**: Zamonaviy PT Sans shrifti va universitet brend ranglari uyg'unligi.
*   **Fully Responsive**: Barcha turdagi qurilmalar (Smartfon, planshet, noutbuk) uchun optimallashtirilgan.
*   **Interactive UI**: Foydalanuvchi bilan jonli muloqot qiluvchi animatsiyalar va interaktiv elementlar.

### 🏢 Universitet Tuzilmasi
*   **Interactive Structure**: Markazlar, bo'limlar va kafedralarning shaffof iyerarxik tuzilmasi.
*   **Leadership**: Oliy ma'lumotli rahbariyat va malakali o'qituvchilar haqida batafsil ma'lumotlar.

### 💻 Raqamli Ekotizim
Platforma quyidagi akademik tizimlar bilan integratsiyalashgan:
*   🚀 **HEMIS**: Oliy ta'lim jarayonlarini boshqarish axborot tizimi.
*   📚 **LMS Platformasi**: Masofaviy o'quv materiallari va topshiriqlari bazasi.
*   📖 **ARM (Elektron Kutubxona)**: Boy va zamonaviy kitoblar hamda ilmiy maqolalar to'plami.
*   📝 **E-Hujjat**: Ichki ma'muriy boshqaruv va hujjat almashinuvi.

---

## 🛠️ Ishlatilgan Texnologiyalar

*   **UI Framework**: [React.js 18](https://reactjs.org/)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Font System**: Google Fonts (PT Sans)
*   **Animation**: Custom CSS Transitions & Framer Motion logic

## 📂 Loyiha Strukturasi

```bash
jbnuu-front-design/
├── public/                 # Statik fayllar va rasmlar
│   └── images/             # Loyiha vizual resurslari
├── src/
│   ├── components/         # UI Komponentlar (Hero, News, Structure, etc.)
│   ├── data/               # Universitet ma'lumotlari ombori
│   ├── assets/             # Komponent ichidagi statik resurslar
│   ├── App.jsx             # Asosiy yo'naltirish (Routing)
│   ├── index.css           # Global dizayn va Tailwind konfigi
│   └── main.jsx            # Entry point
├── index.html              # Asosiy HTML5 skeletonu
└── package.json            # Loyiha sozlamalari va paketlar
```

## 🚀 Ishga Tushirish

1.  **Loyiha klonlash**:
    ```bash
    git clone https://github.com/Tamurbek/jbnuu-new.git
    cd jbnuu-front-design
    ```

2.  **Kutubxonalarni o'rnatish**:
    ```bash
    npm install
    ```

3.  **Loyiha ishga tushirish (Dev)**:
    ```bash
    npm run dev
    ```

4.  **Production build**:
    ```bash
    npm run build
    ```
